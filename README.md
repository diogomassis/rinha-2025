# 🐔 Rinha 2025

Implementação do desafio Rinha de Backend 2025 em Golang, priorizando alto desempenho, concorrência eficiente e mecanismos resilientes de fallback para processadores de pagamento.

Para acessar o desafio, visite: [Rinha de Backend 2025](https://github.com/zanfranceschi/rinha-de-backend-2025).

## Descrição do Projeto

Este projeto contém o código-fonte do desafio Rinha de Backend 2025, que consiste em desenvolver uma aplicação backend que intermedie solicitações de pagamentos para um serviço de processamento de pagamentos, chamado `Payment Processor`.
